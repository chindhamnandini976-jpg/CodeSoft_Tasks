# Task 4 – E-commerce Website

## 📌 Project Overview
This project was created as part of the **CodSoft UI/UX Internship**.

The objective of this task was to design a modern, user-friendly, and visually appealing e-commerce website interface. The design focuses on providing a seamless shopping experience by enabling users to browse products, search for items, view product details, and complete purchases with ease.

---

## 🎯 Objective
- Design an attractive and intuitive e-commerce website.
- Create a smooth shopping experience.
- Organize products into categories.
- Improve navigation and accessibility.
- Apply modern UI/UX design principles.

---

## 🛠️ Tools Used
- Canva / Figma
- Icons and Illustrations
- Google Fonts

---

## ✨ Features
- Modern Homepage Design
- Navigation Bar
- Search Bar
- Product Categories
- Featured Products Section
- Product Cards with Images, Prices, and Ratings
- Shopping Cart Icon
- Wishlist Option
- Special Offers & Discounts
- Responsive Layout
- Clean Footer with Contact Information
