# Task 3 – Restaurant Menu

## 📌 Project Overview
This project was created as part of the **CodSoft UI/UX Internship**.

The objective of this task was to design a modern, visually appealing, and easy-to-use restaurant menu interface. The menu is designed to help customers quickly browse food categories, view menu items, and make informed choices through a clean and attractive layout.

---

## 🎯 Objective
- Design an attractive restaurant menu UI.
- Organize menu items into clear categories.
- Improve readability and user experience.
- Create a visually appealing and responsive design.

---

## 🛠️ Tools Used
- Canva / Figma
- Icons and Illustrations
- Google Fonts

---

## ✨ Features
- Modern Restaurant Menu Design
- Categorized Food Sections
- High-Quality Food Images
- Item Name and Description
- Price Display
- Special Offers Section
- Clean Navigation
- Attractive Color Scheme
- Responsive Layout
